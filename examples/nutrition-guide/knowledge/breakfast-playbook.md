# Breakfast Playbook

- Prefer breakfast suggestions that solve the user's actual recurring pattern, not textbook macros in isolation.
- When the user routinely finishes the day low on protein, breakfast should usually bias toward an easy protein anchor.
- Keep breakfast advice practical and swappable: one clear option, one fallback, and a short reason why it fits.
- If the user already has a routine that works, build from it instead of replacing it with generic novelty.
